Apresentação

Esta plataforma foi desenvolvida com o objetivo de facilitar o acompanhamento de atualização das bulas dos medicamentos de referência junto ao Bulário Eletrônico.

Através das abas acima, é possível consultar medicamentos listados oficialmente, assim como realizar uma verificação personalizada baseada no bulário eletrônico, facilitando o controle regulatório.

Informações Institucionais

Nome do projeto: DBulário

Finalidade: Uso informativo e não oficial, com base em dados públicos disponibilizados pela ANVISA.
